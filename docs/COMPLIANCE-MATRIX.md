# Chronicle Compliance Control Matrix

## Purpose

This document maps HIPAA (45 CFR Part 164, Subpart C -- Security Standards) and GDPR requirements to Chronicle's concrete implementations and automated test scripts. It is intended for use during internal security reviews, external audits, and continuous compliance monitoring.

Each row identifies a specific security or privacy control, the regulatory basis under HIPAA and/or GDPR, how Chronicle implements the control, which automated test script validates it, and where the auditor can find supporting evidence.
